# SvelteThree — Personal Resume & Portfolio Site

## Overview

A clean, minimalist personal resume and portfolio site built with modern web technologies. Designed to showcase professional work, skills, and experience with a focus on simplicity and performance.

## Goals

- **Professional Presence** — ✅ Create a polished online presence that reflects technical skills and professionalism
- **Performance** — ✅ Fast, lightweight site optimized for Core Web Vitals and SEO (Lighthouse 99/100)
- **Simplicity** — ✅ Minimalist design (white, clean aesthetic) that emphasizes content over decoration
- **Maintainability** — ✅ Easy to update, extend, and keep fresh with new projects and experience

## Technical Stack

| Component | Technology | Version |
|-----------|-----------|---------|
| Framework | SvelteKit | 2.57.0 |
| Language | Svelte | 5.55.2 |
| Styling | Tailwind CSS | Latest |
| TypeScript | TypeScript | 6.0.2 |

| Deployment | Vercel | Adapter-auto |

## Design Principles

- **Minimalist**: White background, slate gray typography, minimal decoration
- **Modern**: Svelte 5 runes, utility-first CSS, semantic HTML
- **Accessible**: Semantic HTML, ARIA labels, keyboard navigation
- **Mobile-First**: Responsive design that works on all devices
- **Fast**: Optimized for production, minimal JS, static-first where possible

## Target Audience

Potential employers, collaborators, and professional contacts seeking to learn about the developer's skills, experience, and previous work.

## Scope (All Complete)

- ✅ Homepage/hero section with CTAs
- ✅ About/bio section with professional photo and metrics
- ✅ Experience/work history (3 positions with achievements)
