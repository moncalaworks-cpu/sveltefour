## Notes

- **Photos**: Keep file sizes reasonable (compress images before using)
- **Links**: Use full URLs (https://...)
- **Descriptions**: Be specific and impact-focused, not just task descriptions
- **Dates**: Use consistent format (Jan 2022, Feb 2023, etc.)
- **Tech Stack**: List technologies relevant to the project (not everything you know)
